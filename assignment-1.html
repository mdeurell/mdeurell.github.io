Assignment 1

Data collection and basic visualization

The first assignment focuses on data collection, basic statistical analysis and simple visualizations. Learn to work with CSV files, process raw data and create initial visualizations with static plots.

Download notebook

Assignment tasks and outputs

Assignment 1.1 - Temporal overview

Build a yearly overview of incidents for each personal focus crime across the full combined dataset. The plot should cover at least 2003-2025 and clearly show long-term movement in activity. Use labels, legend, and title, then interpret major shifts such as spikes, drops, or COVID-era effects.

Assignment 1.2 - District crime profiles

Compute conditional crime ratios for each police district to compare local crime profiles with city-wide baselines. Visualize over- and under-representation across districts and focus crimes in one comparable chart view. Select one standout district and explain which geographic or demographic factors might drive the pattern.

Assignment 1.3 - Visualizing distributions

Recreate core distribution views from DAOST using your own crime data: jitter plots, QQ/probability plots, and box-plot style time-of-day comparisons. Use these visuals to assess data precision, normality deviations, and whether crime types show distinct day/night rhythms.

Assignment 1.4 - Spatial power law

Test whether your most frequent focus crime is spatially concentrated by counting incidents in ~100m grid cells. Compare the distribution on linear and log-log axes to evaluate power-law behavior. Use the result to discuss hotspot concentration and limits of neighborhood averages.

Assignment 1.5 - Regression and correlation

Build 168-hour weekly vectors for at least four focus crimes and compare each pair in a scatterplot matrix. Add closed-form linear regression lines and R2 scores to quantify shared weekly rhythms. Use the strongest and weakest pairs to discuss which crime types move together and why.
