Assignment 1

Data collection and basic visualization

The first assignment focuses on data collection, basic statistical analysis and simple visualizations. Learn to work with CSV files, process raw data and create initial visualizations with static plots.

Download notebook

Assignment tasks and outputs

Assignment 1.1 - Temporal overview

The plot aggregates yearly incident counts from 2003 to 2025 for selected focus crimes. It highlights long-term trends with clear declines in some categories and periodic peaks in others. The data view makes it easier to compare how crime activity evolves across categories over time.

Assignment 1.2 - District crime profiles

This output compares district-level crime composition against the city-wide average pattern. Relative differences show where specific crime types are overrepresented or underrepresented. The chart helps identify neighborhood signatures and supports location-specific interpretation.

Assignment 1.3 - Visualizing distributions

The figures summarize how incident times and counts are distributed across the dataset. Jitter, QQ, and box-style views reveal spread, skewness, and deviations from simple normal assumptions. Together, they show that crime timing often follows structured patterns rather than random noise.

Assignment 1.4 - Spatial power law

Incident counts are grouped into small spatial cells to measure concentration across the city. Most cells contain few events while a limited set of hotspots contain many incidents. The linear and log-log views together illustrate heavy-tail behavior in the spatial distribution.

Assignment 1.5 - Regression and correlation

Weekly 168-hour vectors are compared pairwise to quantify similarity between crime rhythms. Regression lines and R2 scores show which categories co-vary strongly and which evolve independently. The static and interactive views support both quick overview and detailed point-level inspection.
